# Домашнее задание по теме
## «Работа с серверами и протокол удалённого управления SSH»

---

## Цель работы
Освоить базовые навыки работы с SSH:
- установка и настройка SSH-сервера;
- подключение к удалённому серверу;
- передача файлов по SCP;
- настройка безпарольного доступа с помощью SSH-ключей.

---

## Исходные данные

- Сервер: Linux (Ubuntu)
- IP-адрес сервера: `192.168.1.75`
- Пользователь сервера: `vlad`
- Клиентская ОС: Windows
- Среда: Git Bash / PowerShell
- Используемые утилиты: `openssh-server`, `ssh`, `scp`, `ssh-keygen`

---

## Кейс 1. Установка и настройка SSH-сервера

### Установка SSH-сервера
```bash
sudo apt update
sudo apt install openssh-server -y
```

### Проверка статуса сервиса
```bash
sudo systemctl status ssh
```

### Результат
SSH-сервер успешно установлен и запущен.  
Статус сервиса: **active (running)**.

---

## Кейс 2. Подключение к удалённому серверу

### Команда подключения
```bash
ssh vlad@192.168.1.75
```

### Результат
Выполнено успешное подключение к серверу по SSH с использованием логина и IP-адреса.

---

## Кейс 3. Копирование файлов с помощью SCP

### Копирование файла с Windows на Linux-сервер
```bash
scp "C:\Users\сникерок\Documents\ci cd\t1.txt" vlad@192.168.1.75:/home/vlad/
```

### Копирование файла с Linux-сервера на Windows
```bash
scp vlad@192.168.1.75:/home/vlad/t1.txt "C:\Users\сникерок\Documents\"
```

### Результат
Файл успешно передан на сервер и получен обратно на локальную машину.

---

## Кейс 4. Работа с SSH-ключами

### Генерация SSH-ключей
```bash
ssh-keygen
```

Была сгенерирована пара ключей типа **ED25519**.

### Добавление публичного ключа на сервер
```bash
cat ~/.ssh/id_ed25519.pub | ssh vlad@192.168.1.75 "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"
```

### Настройка прав доступа на сервере
```bash
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys
```

### Проверка входа без пароля
```bash
ssh vlad@192.168.1.75
```

### Результат
Настроен безпарольный доступ к серверу по SSH с использованием ключей.
